### 自定义底部导航栏+Lottie图标动画
本项目是已androidx + lottie 最新版本构建
如果是旧版本（support库）请使用lottie 2.7

setCustomSelectedPosition 设置默认选中按钮 如果设置默认选中 则在点击的时候就需要手动执行这句

#### 架构
rigger androidx 库 一个AOP管理fragment的
lottie  动画库
